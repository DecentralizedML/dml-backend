# Decentralized ML (back-end)

Back-end NodeJS server for the Decentralized ML project.

## Getting Started

### Prerequisites

* NodeJS **10.6.0**

### Setup

Copy the dotenv file:

```bash
$ cp .env.sample .env
```

Install dependencies:

```bash
$ yarn
```

## Running the server

Run the server with:

```bash
$ node app.js
```

The server will be accessibble via http://0.0.0.0:3000/ by default.

## Running the tests

```bash
$ yarn test
```
